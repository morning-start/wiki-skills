# 文档一致性检查清单

## 使用说明

本清单用于系统性地检查文档一致性，涵盖 7 大类检查内容。根据文档类型和重要性，选择相应的检查项。

**适用场景**：
- 文档生成后的质量检查
- 文档更新后的一致性验证
- 版本发布前的文档审核
- 项目里程碑评审

**使用方法**：
1. 根据文档类型选择检查类别
2. 逐项检查并标记状态（✅ 通过 / ❌ 未通过 / ⚠️ 部分通过）
3. 记录问题和修复建议
4. 跟踪问题修复

## 附录：快速检查脚本

### 检查版本号一致性

```bash
# 检查 SKILL.md 和 CHANGELOG.md 版本号
grep -h "version:" SKILL.md
grep -h "^## \[" CHANGELOG.md | head -1
```

### 检查日期格式

```bash
# 检查 CHANGELOG 中的日期格式
grep -E "^\d{4}-\d{2}-\d{2}" CHANGELOG.md
```

### 检查文档命名

```bash
# 检查核心文档命名
ls -la | grep -E "(README|CHANGELOG|ROADMAP|ARCHITECTURE)"
```
