# 智能绘图策略

本文档定义文档中图示的智能选择策略，根据内容复杂度自动选择 TXT 字符图或 Mermaid 图表。

## 决策流程

```
绘图需求
   │
   ▼
┌─────────────────────────────┐
│ 分析图类型和复杂度           │
└──────────┬──────────────────┘
           │
    ┌──────┼──────┐
    ▼             ▼
  简单          复杂
    │             │
    ▼             ▼
  TXT图      Mermaid图
```

## 复杂度判定规则

### 架构图

| 维度 | 简单 (TXT) | 复杂 (Mermaid) |
|------|-----------|----------------|
| 组件数量 | < 5 | ≥ 5 |
| 层级深度 | ≤ 2 | > 2 |
| 连线数量 | < 8 | ≥ 8 |
| 特殊元素 | 无 | 有负载均衡/网关等 |

### 时序图

| 维度 | 简单 (TXT) | 复杂 (Mermaid) |
|------|-----------|----------------|
| 参与者数量 | ≤ 2 | > 2 |
| 消息数量 | ≤ 5 | > 5 |
| 循环/分支 | 无 | 有 |

### ER 图

- **始终使用 Mermaid**，TXT 难以清晰表达关系

### 状态图

| 维度 | 简单 (TXT) | 复杂 (Mermaid) |
|------|-----------|----------------|
| 状态数量 | ≤ 5 | > 5 |
| 转换数量 | ≤ 8 | > 8 |
| 并行状态 | 无 | 有 |

### 流程图

| 维度 | 简单 (TXT) | 复杂 (Mermaid) |
|------|-----------|----------------|
| 分支数量 | ≤ 2 | > 2 |
| 循环结构 | 无 | 有 |
| 子流程 | 无 | 有 |

## TXT 图规范

### 组件样式

```
┌─────────────┐
│   组件名称   │
└─────────────┘
```

- 宽度：13 字符（不含边框）
- 高度：3 行（含边框）
- 组件名居中显示

### 连线样式

```
┌─────┐    HTTPS    ┌─────┐
│  A  │ ─────────► │  B  │
└─────┘             └─────┘
```

- 实线：`───►` 单向，`◄───►` 双向
- 标注：连线上部标注协议/关系
- 转弯：使用 `│` 垂直连接

### 数据库样式

```
┌─────────────────────────┐
│      PostgreSQL          │
│  ┌─────┐  ┌─────┐       │
│  │users│  │orders│      │
│  └─────┘  └─────┘       │
└─────────────────────────┘
```

## Mermaid 规范

### 必须包含配色

```mermaid
%%{
  init: {
    "theme": "base",
    "themeVariables": {
      "primaryColor": "#e3f2fd",
      "primaryTextColor": "#1a237e",
      "primaryBorderColor": "#1976d2",
      "lineColor": "#546e7a",
      "secondaryColor": "#f3e5f5",
      "tertiaryColor": "#e8f5e9"
    }
  }
}%%
```

### 配色方案

#### 亮色主题（默认）

```yaml
primaryColor: "#e3f2fd"
primaryTextColor: "#1a237e"
primaryBorderColor: "#1976d2"
lineColor: "#546e7a"
secondaryColor: "#f3e5f5"
tertiaryColor: "#e8f5e9"
```

#### 暗色主题

```yaml
primaryColor: "#1a237e"
primaryTextColor: "#e3f2fd"
primaryBorderColor: "#64b5f6"
lineColor: "#90a4ae"
secondaryColor: "#4a148c"
tertiaryColor: "#1b5e20"
```

### 文字可读性

- 文字颜色与背景色对比度 ≥ 4.5:1（WCAG AA）
- 避免使用浅色文字在浅色背景上
- 节点文字优先使用简短描述

### 图标使用

- 可选使用 emoji 增强可读性
- 常用图标：
  - 🖥️ 客户端
  - 🚪 网关
  - 👤 用户服务
  - 📦 订单服务
  - 💳 支付服务
  - 📊 数据库
  - ⚡ 缓存
  - 📮 消息队列

## 图类型选择矩阵

| 场景 | 推荐图类型 | 说明 |
|------|-----------|------|
| 系统架构 | flowchart | 组件关系 |
| 用户登录流程 | sequenceDiagram | 交互时序 |
| 数据库设计 | erDiagram | 实体关系 |
| 订单状态机 | stateDiagram-v2 | 状态转换 |
| CI/CD 流程 | flowchart | 流程步骤 |
| 需求分解 | mindmap | 层次结构 |
| 项目计划 | gantt | 时间规划 |

## 注意事项

- 图必须有标题说明用途
- TXT 图注意对齐，使用等宽字体
- Mermaid 图测试渲染效果
- 复杂图可拆分为多个子图
- 图与正文内容保持一致
